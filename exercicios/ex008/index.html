Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semantica)

Nessa frase temos um termo em dastaque usando a tag STRONG (semantica)

Italica / Enfase

Nesta frase temos um termo em italico usando a tag I (não semantico)

Nesta frase temos um termo em enfase usando a tag EM (semantica)

Texto marcado

Podemos criar tambem um texto marcado usando a tag MARK

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse paragrafo. Com as tags SMALL e BIG.

Texto deletado

podemos marcar um texto como excluida para indicar que ele deve ser lido mas não consierado.Tag DEL.

Texto inserida

Podemos marcar um texto com inserido para dar uma enfase e indicar que ele foi adicionado depois. Tag INS (semantica)

Existe tambem o sublinhado com a tag U (não semantica)

Texto sobrescrito

Para inserir cisas do tipo x20+3=9 com a tag SUP.

Texto Subscrito

Para inserir coisas do tipo H2O co a tag SUB.
